# Misc. node shit

## Update your dependencies

Install npm chekc updates:
```
npm install npm-check-updates -g
```

In your package.json folder:

```
npm-check-updates
npm-check-updates -u
```

You might need to delete your node_modules or update your npm (`brew update npm`).
